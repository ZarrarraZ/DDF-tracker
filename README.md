# Die drei ??? – Meine Sammlung

Eine PWA zum Tracken aller Die drei ??? Hörspielfolgen.

## GitHub Pages Setup

1. Dieses Repository auf GitHub erstellen (Public)
2. Alle Dateien hochladen
3. **Settings → Pages → Source: main branch → Save**
4. Nach ~2 Minuten erreichbar unter: `https://DEINNAME.github.io/REPO-NAME/`
5. In Chrome auf Android: **Menü → App installieren**

## Features

- 237 normale Folgen + 24 Spezialfolgen
- Folgen als gehört markieren mit Datum
- „Nochmal gehört" – vollständiger Hörverlauf
- Bewertung 1–5 in halben Schritten
- Persönliche Tags und Notizen
- Zufälliger Folgen-Finder
- Statistiken: Kalender, Hörstunden, Top-Folgen
- Spotify-Link pro Folge
- Export/Import als JSON-Backup
- Cover automatisch von iTunes geladen
- Funktioniert offline nach erstem Laden
